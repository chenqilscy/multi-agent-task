# 🎉 旅程5：会客模式

## 概述

有客人来访时，系统调整家中氛围，包括灯光、音乐和温度的协调设置。

## 涉及Agent

- **LightingAgent**: 灯光氛围调节
- **MusicAgent**: 背景音乐播放
- **ClimateAgent**: 温度舒适度调节

## 复杂度

**L3** - 需要三个Agent协作完成场景设置
