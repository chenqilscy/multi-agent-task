## 🎯 旅程目标

为用户提供舒适的起床体验，通过查询天气、调节温度、控制灯光等，让用户自然从睡眠状态过渡到清醒状态。

## 📊 能力覆盖

- **涉及Agent**: WeatherAgent, ClimateAgent, LightingAgent, MusicAgent
- **核心能力**: 天气查询、穿衣建议、温度调节、灯光渐亮、音乐播放
- **异常处理**: 服务故障降级、设备离线提示
