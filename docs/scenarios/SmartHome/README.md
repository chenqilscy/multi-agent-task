# SmartHome智能家居场景用例

本目录包含智能家居场景的36个用例，涵盖8个用户旅程。

## 🎯 场景概述

SmartHome场景展示了多Agent协作完成家居自动化控制的能力，包括：

- **WeatherAgent**: 查询天气、提供穿衣出行建议
- **ClimateAgent**: 空调、温度调节
- **LightingAgent**: 灯光开关、亮度调节
- **MusicAgent**: 音乐播放控制
- **TemperatureHistoryAgent**: 温度历史记录
